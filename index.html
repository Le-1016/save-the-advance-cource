高専教育への緊急問いかけ / URGENT ALERT

専攻科が、 消えた。 全国57校のうち、1校だけで。

大阪公立大学高専で、専攻科が廃止されました。
しかしこれは専攻科だけの問題ではない——
高専本科そのものの存在意義が、構造的に薄まっていく。
全国の高専関係者の皆さんに、問いかけます。

発起人：大阪公立大学工業高等専門学校 現役生

// Section 01 — What happened

2024年度、大阪公立大学高専の
専攻科が事実上廃止されました。

⚠ 全国57高専中、専攻科を廃止したのはこの1校のみ。高専教育の歴史において前例のない事態です。

2021年1月に示された高専改革方針により、2024年4月入学生を最後に専攻科の学生募集が停止されました。新専攻科の設置予定はありません。

廃止の理由として挙げられたのは、大阪公立大学との「教育内容の重複」と、大学編入枠の拡大による代替です。しかし——この「代替」に、具体的な保証はあるのでしょうか？

参照資料①：高専改革の背景

大阪公立大学高専の改革方針が示されている大阪府の公式ページ。専攻科廃止の経緯と、高専と大阪公立大学の統合的位置づけが確認できます。この特殊な法人一体構造が、今回の廃止を可能にした制度的背景です。

大阪府：大阪公立大学高専改革について →
57
全国の国公私立高専数（2025年現在）

専攻科を廃止した高専は、この1校のみ。
高校でも大学でもない、高専という特殊な教育機関。その中でさらに極めて異例のこの事態は、全国に「前例」を刻みます。

// Section 02 — The false substitute

「大学編入で代替できる」——
保証はどこにも書かれていない。

法人側は「専門性を高めたい学生は大学編入で対応する」としています。しかし、大阪公立大学工学部の編入学募集要項を確認しても、大阪公立大学高専生への内部進学の優遇・保証は明記されていません。

参照資料②：大学編入に内部進学の保証はあるか

大阪公立大学工学部の編入学試験情報ページ。外部の高専生・大学生と同じ試験で競合する構造になっており、大阪公立大学高専生への特別枠や内部進学の保証は確認できません。「代替できる」という主張の根拠を、自分の目で確かめてください。

大阪公立大学：編入学試験情報を確認する →
専攻科（廃止前）
  • 本科5年の積み上げを前提とした一貫指導
  • 同じ教員のもとで研究を継続
  • 少人数・PBL型の実践的研究
  • 高専教育の「完成形」としての位置づけ
  • 内部競争のみで進学可能（外部と競合しない）
  • 修了で「学士」相当の称号取得
大学編入（代替案）
  • 大学の教育設計・文化に合わせる必要がある
  • 研究テーマ・指導教員が変わる
  • 外部高専生・大学生との競争枠
  • 高専教育との連続性が保証されない
  • 情報収集・受験対策が新たに必要
  • 内部優遇の明記なし（要項上は平等競争）

「代替できる」と言うなら、内部進学の保証を明示すべきだ。それがない限り、廃止の正当性は成立しない。

// Section 03 — The credential problem

本科の称号すら未完全なのに、
「大学学士で成り立つ」と廃止する矛盾。

高専本科を修了しても、授与されるのは「準学士」相当の称号です。大学の「学士」とは法的・社会的に異なります。その状態のまま「大学に編入すれば学士が取れるから専攻科は不要」と廃止するのは——

⚠ 「高専教育は非効率であり、大学学士授与機関として正式な大学のほうが優れている」と、法人自身が宣言しているに等しい。

では、なぜ高専が存在するのか。高専は「高校でも大学でもない」独自の教育機関として、実践的技術者教育を担ってきました。その価値を自ら否定する形で専攻科を廃止することは、高専教育の根幹への問いかけです。

// Section 04 — The bigger threat

専攻科だけではない。
本科の存在意義が、構造的に薄まっている。

私はこれが「偶然の積み重ね」とは思えません。以下の要素を並べると、一つの方向性が見えてきます。

危機の構造：段階的に積み上がる圧力
01
大阪府の私立高校無償化——高校から直接、大阪公立大学を目指すルートが拡大。高専を経由しなくても大学進学できる環境が整備されている。
▼
02
募集定員160人中120人が女子枠・推薦枠——一般枠はわずか40人。外部との競争激化が進み、高専本科への入学難易度が上昇。「高専に入る意味」が問われる構造になっている。
▼
03
2027年、中百舌鳥キャンパスへ移転——移転先のキャンパスは大阪公立大学のもの。高専の独自性・物理的独立性が失われ、大学の付属機関化が進む恐れがある。
▼
04
専攻科廃止（2024年〜）——「大学で代替できる」という論理で、高専独自の上位教育を消去。高専本科を「大学進学のための予備過程」に位置づける方向性が見える。
▼
05
仮に本科が廃止されても——キャンパスは大阪公立大学のもの、教員・設備は大学と共有、学生は大学に編入できる。廃止しても困る主体が存在しない構造が既に出来上がっている。

これは偶然の積み重ねか、それとも設計された構造変化か——全国の高専関係者に問いたい。

高専は高校ではありません。全国57校が持つ独自の教育文化と実績は、日本の技術教育の根幹です。大阪公立大学高専での動きが「前例」となれば、他校への波及は避けられません。高専という教育機関でこのようなことが起きると、その特殊な成り立ちゆえに全国で目立ちます。だからこそ、今、改めて問い直さなければならない。

// Section 05 — Three questions

全国の高専関係者へ、3つの問い。

QUESTION 01

「大学学士で代替できるから専攻科は不要」——この論理が認められるなら、高専本科の独自価値も同様に否定されないか？

QUESTION 02

キャンパス移転・定員構造の変化・専攻科廃止という三重の変化は、高専の段階的な大学組み込みへの布石ではないか？

QUESTION 03

大阪公立大学高専の事例が「前例」となることで、全国の他の高専でも同様の「効率化」「統合」が正当化されるリスクはないか？

⚠ これは大阪だけの問題ではありません。高専という教育機関の将来に関わる、全国共通の問いです。
